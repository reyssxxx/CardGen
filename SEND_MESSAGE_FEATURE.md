# Функция "Отправить Сообщение Классу"

## ✅ Реализовано

Добавлена полная функциональность отправки сообщений всему классу для учителей.

## 📋 Как Работает

### Шаг 1: Нажать кнопку "📢 Отправить сообщение классу"

Учитель видит Reply кнопку в главном меню учителя.

### Шаг 2: Выбрать класс

Показывается список классов учителя (из config.json).

### Шаг 3: Ввести текст сообщения

- Минимум: 5 символов
- Максимум: 4000 символов
- Обычный текст

### Шаг 4: Автоматическая отправка

Бот автоматически:
1. Находит всех учеников класса из `students.json`
2. Ищет их в базе данных по имени
3. Отправляет каждому зарегистрированному ученику

## 📊 Формат Сообщения

Ученики получают:

```
📢 Сообщение от учителя

[Текст сообщения]

—
👨‍🏫 Учитель: @username
```

## 🔧 Техническая Реализация

### Файлы

**handlers/teacher_handlers.py:**
- `reply_button_send_message()` - инициализация
- `send_message_class_selection()` - выбор класса
- `send_message_text_input()` - ввод и отправка

**handlers/states.py:**
- `TeacherSendMessage` - FSM состояния

**database/user_repository.py:**
- `get_user_by_name()` - поиск ученика по ФИО (обновлён с `role`)

### FSM Состояния

```python
class TeacherSendMessage(StatesGroup):
    waiting_for_class = State()      # Ожидание выбора класса
    waiting_for_message = State()    # Ожидание текста сообщения
```

### Процесс Отправки

```python
for student_name in students:
    user = user_repo.get_user_by_name(student_name)
    if user and user['role'] == 'student':
        await bot.send_message(chat_id=user['ID'], text=full_message)
        sent_count += 1
```

## 🔍 Обработка Ошибок

### Не найден ученик
Если ученик не зарегистрирован в боте:
- Не отправляется сообщение
- `failed_count` увеличивается
- Логируется в консоль

### Ошибка отправки
Если Telegram API вернул ошибку:
- Ловится exception
- `failed_count` увеличивается
- Логируется в консоль

### Сообщение слишком короткое/длинное
Валидация:
- < 5 символов: "❌ Сообщение слишком короткое"
- > 4000 символов: "❌ Сообщение слишком длинное"

## 💡 Особенности

### Только зарегистрированные ученики
- Отправка только ученикам, которые уже запустили бота
- Незарегистрированные не получат сообщение
- Отображается в статистике

### Проверка роли
```python
if user and user['role'] == 'student':
    # Отправляем только ученикам, не учителям
```
