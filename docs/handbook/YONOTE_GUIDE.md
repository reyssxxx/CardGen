# Как загрузить хендбук в YoNote

## 1. Регистрация и создание пространства

1. Зайди на [yonote.ru](https://yonote.ru) и зарегистрируйся.
2. Создай новое **пространство** (Space) — например, «Лицей ЮФУ».
3. Внутри пространства будут страницы — по одной на каждый раздел хендбука.

---

## 2. Структура страниц

Создай следующие страницы в таком порядке:

```
📖 Хендбук лицеиста       ← главная, её делаешь первой
  🏫 О лицее
  ⏰ Распорядок дня
  📋 Правила и устав
  🎓 Учёба и оценки
  🎉 Мероприятия
  💬 Психологическая поддержка
  📞 Контакты
  🚀 Первый день
  ❓ Частые вопросы
```

Подстраницы создаются через **«+ Добавить страницу»** внутри родительской.

---

## 3. Как вставить контент

YoNote поддерживает вставку Markdown напрямую:

1. Открой нужную страницу.
2. Нажми в любое место на странице.
3. Скопируй нужный блок из файла `HANDBOOK_CONTENT.md`.
4. Вставь — YoNote автоматически распознает Markdown.

> Если таблицы или заголовки не распознались — попробуй вставить через **Ctrl+Shift+V** (вставка без форматирования, потом YoNote сам разберёт).

---

## 4. Как сделать красивую главную с карточками

Простая `@ссылка` выглядит как синий текст без описания — не очень. Чтобы сделать как в хендбуке грантовика (иконка + заголовок + описание + ссылки в колонках):

### Шаг 1 — создай колонки

Напиши `/columns` или перетащи блок вправо от другого блока — появятся две колонки. Так для каждой пары карточек.

### Шаг 2 — наполни каждую колонку

Внутри колонки по очереди добавь:
1. Эмодзи (просто текстом: `🏫`)
2. Жирный заголовок: `**О лицее**`
3. Обычный текст-описание: `История, профили обучения, педсостав`
4. Пустая строка
5. Ссылки через `@` — каждая на отдельной строке:
   - `@О лицее` (ссылка на всю страницу)
   - или `/` → выбери **«Упоминание страницы»**

### Как это выглядит в итоге

```
[Колонка 1]                        [Колонка 2]

🏫                                 ⏰
**О лицее**                        **Распорядок дня**
История, профили, педсостав        Расписание звонков, перемены

@О лицее                           @Распорядок дня
@Педагогический состав
@Контакты и адрес
```

### Ссылка на конкретный раздел страницы

YoNote пока не поддерживает якоря (`#раздел`) — ссылка ведёт только на страницу целиком. Поэтому на главной достаточно одной `@ссылки` на каждую подстраницу.

---

## 5. Настройка доступа

После того как всё заполнено:

1. Зайди в настройки пространства → **«Доступ»** или **«Поделиться»**.
2. Выбери **«Доступ по ссылке»** → **«Просмотр»**.
3. Скопируй ссылку — она будет вида `https://yonote.ru/share/xxxxx`.
4. Вставь эту ссылку в бот вместо старой ссылки на GitHub Pages.

---

## 6. Замена ссылки в боте

Открой файл `keyboards/student_keyboards.py`, найди строку:

```python
HANDBOOK_URL = "https://reyssxxx.github.io/CardGen/handbook/"
```

Замени URL на ссылку из YoNote.
