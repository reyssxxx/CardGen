## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

**Важно**: Установка EasyOCR может занять несколько минут, так как загружаются предобученные модели.

### 2. Настройка .env файла

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_ID=123456789,987654321
```

- `BOT_TOKEN` - токен вашего бота от @BotFather
- `ADMIN_ID` - ID администраторов через запятую

## Тестирование OCR (без бота)

Создан тестовый скрипт для проверки OCR без запуска бота:

```bash
python test_ocr.py <путь_к_фото_журнала>
```

Пример:
```bash
python test_ocr.py data/uploaded_photos/journal_test.jpg
```

Скрипт выведет:
- Распознанный класс
- Распознанные даты
- Список учеников с оценками
- Количество оценок готовых для сохранения в БД
- Предупреждения (если есть)
- Debug информацию

## Использование функционала учителя

### Регистрация учителя

1. Отправьте боту `/start`
2. Выберите "Я преподаватель"
3. Введите ФИО (например: "Иванов Иван Петрович")

**Важно**: У учителя ОБЯЗАТЕЛЬНО должен быть установлен username в Telegram!

### Загрузка фото журнала

**Команда**: `/photo`

**Workflow**:
1. Выбор предмета из списка ваших предметов
2. Выбор класса (можно выбрать автоопределение)
3. Отправка фотографии журнала
4. Ожидание обработки (10-30 секунд)
5. Проверка распознанных дат
   - Если даты верны: "✅ Все даты верны"
   - Если нужно исправить: нажмите на номер даты
6. Проверка оценок каждого ученика
   - Просмотр оценок по датам
   - Редактирование при необходимости (нажмите на столбец)
   - Подтверждение: "✅ Все верно"
7. Финальное подтверждение
   - "✅ Сохранить все оценки" - сохранить в БД
   - "✏️ Продолжить редактирование" - вернуться к проверке
   - "❌ Отменить" - отменить всю загрузку

**Результат**:
- Оценки сохранены в БД
- Фото удалено с сервера
- Ученики получат уведомления (будет реализовано в Этапе 11)

### Рекомендации для фото

Для лучшего распознавания:
- 📸 Фотографируйте при хорошем освещении
- ❌ Избегайте бликов и теней
- 📐 Расположите камеру параллельно странице
- ✔️ Убедитесь что текст четкий и читаемый
- 🔍 Включите в кадр весь журнал (заголовок + таблица)

## Известные проблемы и ограничения

### OCR
- ❗ Рукописный текст распознается хуже печатного
- ❗ Фото под большим углом может плохо распознаваться (но есть коррекция перспективы)
- ❗ Низкая контрастность ухудшает качество распознавания

### Решения
- ✅ Всегда есть возможность отредактировать результаты вручную
- ✅ Предобработка изображений улучшает качество распознавания
- ✅ Валидация со списком учеников помогает находить ошибки
